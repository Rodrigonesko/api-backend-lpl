	Instalação
</h4>
<p align="center">
    Certifique-se de ter o Node.js e o MongoDB instalados em sua máquina
    <p>
        Clone este repositório:
        git clone https://github.com/seu-usuario/api-exemplo.git
    </p>
    <p>
        Acesse o diretório do projeto:
        cd api-exemplo
    </p>
    <p>
        Instale as dependências:
        npm install
    </p>
